> A little less noise. A little more signal.

This project is a playful, scroll-worthy feed for the newest AI products and the buzz around them. Instead of drowning in random updates, it helps you spot what feels genuinely interesting and worth paying attention to.

## Why this exists

The internet is loud. AI launches are everywhere. This project turns that chaos into a calm, ranked feed that highlights the products getting real momentum.

## What it does

- Collects product signals from multiple sources
- Scores each item by engagement and momentum
- Ranks the feed from high buzz to low buzz
- Keeps the experience simple, fast, and easy to explore

## The vibe

Think of it as your soft-launch radar for AI.

- Clean and readable
- Low-stress to explore
- Made for people who want signal without the spam

## Run it locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000

- Next.js
