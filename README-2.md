> Where new AI ideas meet calm, curious attention.

If you love discovering cool things but hate the chaos, this is for you.

This project collects fresh AI product launches and ranks them by buzz so you can quickly see what people are actually talking about.

## What makes it interesting

- It feels more like a discovery feed than a boring dashboard
- It turns scattered signals into one clean experience
- It is built to be simple, readable, and surprisingly addictive to scroll

## How it works

1. Gather product and engagement data
2. Score the buzz with a simple formula
3. Show the most interesting results first
4. Keep the whole experience lightweight and easy to follow

## Why people might like it

Because staying updated should feel a little more like exploring and a little less like doomscrolling.

## Quick start

```bash
npm install
npm run dev
```

## Built with

- Next.js
- Supabase
- TypeScript
- Vitest

## Final note

It is a feed for the curious, the early browsers, and the people who love seeing what is next before everyone else does.
