> The feed that feels like a good late-night scroll.

This project is built for people who want to stay on top of AI products without feeling buried under hype.

It gathers new launches, tracks the buzz, and turns everything into a clean, ranked feed that is easy to skim and hard to ignore.

## What you get

- A clear view of what is gaining attention
- A simple ranking system based on engagement
- A lighter, more human way to discover new products

## Why it is cool

It is not just another list. It is a signal-based discovery experience.

You get the feeling of:

- “Oh wow, this is interesting”
- “Okay, this one is actually popping”
- “I need to know more about that”

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000 and let the feed do its thing.

## Stack

- Next.js
- TypeScript
- Supabase
- Vitest

## Tiny philosophy

Less noise. More curiosity.
