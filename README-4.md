> Low-key iconic. High-signal. Very much worth your attention.

This project is a soft, stylish way to track AI products that are actually creating momentum.

Instead of reading endless updates, you get a feed that surfaces the most interesting launches and the buzz around them in one place.

## What it does

- Watches product momentum
- Scores the energy behind each item
- Sorts everything from most talked-about to least talked-about
- Keeps the experience simple enough to enjoy

## Why it feels different

It is designed for people who want discovery without the stress.

No clutter. No chaos. Just a clean feed that feels a little more thoughtful than the usual internet noise.

## Run locally

```bash
npm install
npm run dev
```

## Built with

- Next.js
- Supabase
- TypeScript
- Vitest

## The real goal

Make AI discovery feel lighter, smarter, and a little more fun.
