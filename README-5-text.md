# AI Product Buzz Feed

Hero image: [docs/assets/hero-welcome.svg](docs/assets/hero-welcome.svg)

> Scroll less. Notice more.

This project is your soft landing space for the AI world.

Instead of chasing every launch and headline, it brings together the products that seem to be picking up real momentum and shows them in one clean, easy-to-browse feed.

## The idea

It is built for curious people who want to stay in the loop without getting buried in hype.

You get:

- a focused view of fresh AI products
- ranked buzz based on engagement
- a smoother, more satisfying discovery experience

## Why it feels good

Because it turns noise into something that feels almost effortless to explore.

You are not reading a giant manual. You are just drifting into the interesting part.

## Get started

```bash
npm install
npm run dev
```

## Stack

- Next.js
- TypeScript
- Supabase
- Vitest
