# AI Product Buzz Feed

Hero image: [docs/assets/hero-welcome.svg](docs/assets/hero-welcome.svg)

> The internet is noisy. This feed is not.

Welcome to a calmer way to discover AI products.

This project collects new launches, tracks their buzz, and turns the whole thing into a feed that feels less like a document and more like a discovery experience.

## What makes it worth a look

- It helps you spot what is actually gaining attention
- It keeps the interface simple and easy to skim
- It makes staying updated feel lighter and more intentional

## The vibe

Think of it as a little corner of the web for people who want to know what is next without getting overwhelmed.

## Run it

```bash
npm install
npm run dev
```

## Built with

- Next.js
- TypeScript
- Supabase
- Vitest
